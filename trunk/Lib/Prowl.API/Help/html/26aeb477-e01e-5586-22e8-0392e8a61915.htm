Priority (Optional)
Type: NuGardt.Prowl.API eProwlPriority
Optional (Default: eProwlPriority.Normal). Default value of 0 if not provided. An integer value ranging [-2, 2] representing:
 
     -2 = Very Low
     -1 = Moderate
      0 = Normal
      1 = High
      2 = Emergency
 
 Emergency priority messages may bypass quiet hours according to the user's settings.
URL (Optional)
Type: System String
Optional. Requires Prowl 1.2 The URL which should be attached to the notification.

 This will trigger a redirect when launched, and is viewable in the notification list.
