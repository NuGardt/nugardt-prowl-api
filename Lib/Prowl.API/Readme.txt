=NuGardt Prowl API v1.0.0.0=
 Author: Kevin 'OomJan' Gardthausen
 Last Update: 2013-06-28

 Visual Basic .NET 4.0 implementation of the Prowl API (http://www.prowlapp.com/api.php). Project includes console client and test project. Runs under Mono (v3) framework.


==Open source==
 Download the source at http://code.google.com/p/nugardt-prowl-api/ or via Subversion (SVN): svn checkout http://nugardt-prowl-api.googlecode.com/svn/trunk/ nugardt-prowl-api-read-only

 Written with Microsoft Visual Studio 2010 in .NET Framework 4.0.
